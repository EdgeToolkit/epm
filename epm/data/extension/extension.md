# 扩展工具





## 自定义扩展原型

如果系统自带的扩展原型不能满足你的需求，用户可以通过自定义原型来实现特定所需。用户自定义的扩展原型通过命令注册后使用，如 MyExtension

```
epm extension register <extension prototype directory>
```

### 目录结构

| 文件名        |      |      |
| ------------- | ---- | ---- |
| prototype.yml |      |      |
| prototype.py  |      |      |
|               |      |      |
